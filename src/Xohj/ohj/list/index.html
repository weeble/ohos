Quick Start

        $('#userlist').ohjlist();
    </script>

Styling

Class Description
.ohjlist Class for the list
.ohjlist-info Base class for top left or right info text
.ohjlist-info-bottom Use with ohjlist-info to position info text on the bottom right of the list item
.ohjlist-info-top Use with ohjlist-info to position info text on the top right of the list item 
       <ul id="userlist" class="ohjlist">
            <li>
                <img src="img/1.jpg" />

            </li>
       </ul>
