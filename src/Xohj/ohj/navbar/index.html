ohj.navbar UI control b0.1

A horizontal bar that contains header text and left and right buttons.



Usage

Quick Start

    <script>
        $('#header').ohjnavbar(
        {
            leftbuttontext : 'Back',
            rightbuttontext : 'Next',
            onleftbuttonpress : function() { },
            onrightbuttonpress : function() { }
        });
    </script>
    ...
    <div id="header">
       Header
    </div>
Header

Options

Option Description Default
title The header text null
leftbuttontext The text of the left button. If unused, then the button will not appear. null
rightbuttontext The text of the right button. If unused, then the button will not appear. null
onleftbuttonpress The function to run when the left button is pressed null
onrightbuttonpress The function to run when the right button is pressed null
height The height of the navbar in css format null
extend Customise the browser discrimination logic to load control extensions null

Public Methods

Method Description
setTitle Set the center text of the nav bar
refresh Recalculates the space available for the header text
