ohj.UI b0.1

Provides UI helper functions



Usage

$.fn.press

Hook up the click or tap event to an DOM element depending on whether the browser is touchable.

    <script>
        $('#pressbtn').press(function() {
            alert('You Pressed me!');
        });
    </script>
    ...
    <button id="pressbtn">Press Me!</button>

$.fn.blockUI / $.fn.unblockUI

Blocks the UI. Useful for modal dialogs and transitions in progress.

    <script>
        $('#blockbtn').press(function() {
            $.fn.blockUI();
            setTimeout(function() {
                $.fn.unblockUI();
            },5000);
        }); 
    </script>
    ...
    <button id="blockbtn">Block UI!</button>

$.fn.template

Generates templated snippets

    <script>
        $.fn.template($('#tplFruit').html(), { rows: 
            [
                { name: 'Orange', quantity: 10 }
                { name: 'Apple', quantity: 20 }
            ]});
    </script>
    ...
    // Template
    <script type='text/html' id="tplFruit">
            <ul>
                <# for(row in rows) { #>
                    <li>There are <#= rows[row].count #> <#= rows[row].name #></li>
                <# } #>
            </ul>
    </script>
