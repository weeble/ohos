ohj.borderlayout UI control b0.1

A layout control that divides the area into top, bottom, left, right and middle regions.



Usage

Quick Start

<script>
    $('#area').ohjborderlayout();
</script>
...
<div id="borderlayout">
    <div class="ohjborderlayout-top">Top</div>
    <div class="ohjborderlayout-main">
        <div class="ohjborderlayout-left">Left</div>
        <div class="ohjborderlayout-middle">Middle</div>
        <div class="ohjborderlayout-right">Right</div>
    </div>
    <div class="ohjborderlayout-bottom">Bottom</div>   
</div>

Options

Option Description Default
height The height of the ohjborderlayout container in css format. null
topheight The height of the top region in css format. null
bottomheight The height of the bottom region in css format. null
leftwidth The width of the left region in css format. null
rightwidth The width of the right region in css format. null
extend Customise the browser discrimination logic to load control extensions null

Styling

Class Description
.ohjborderlayout Class for the borderlayout.
.ohjborderlayout-top Class for the top region.
.ohjborderlayout-bottom Class for the bottom region.
.ohjborderlayout-left Class for the left region.
.ohjborderlayout-right Class for the right region.
.ohjborderlayout-middle Class for the middle region.
.ohjborderlayout-main Class for the region that span the left, right and middle regions.
