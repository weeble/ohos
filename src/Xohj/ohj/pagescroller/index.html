ohj.pagescroller UI control b0.1

A container control which allows it's content to be scrolled.



Usage

Quick Start

    <script>
        $('#content').ohjpagescroller();
    </script>
    ...
    <div id="content">
       ...Content to be scrolled goes here...
    </div>

Dependencies

Dependency Platform Description
jquery 1.7.2 Desktop Core library for desktop browsers
zepto 1.0rc1 Mobile Webkit Core library for android and ios browsers
ohj.pagescroller.pointer Desktop Uses a slim scrollbar to allow content to be scrolled. The mousewheel can also be used to scroll.
ohj.pagescroller.touch Mobile Webkit Mimics a scrollable panel on touch devices. Not ideal for all Android devices.
