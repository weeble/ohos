ohj.util b0.1

Provides util helper functions



Usage

$.fn.uuid

Generates a RFC4122 v4 compliant unique identifier

    <script>
        $('#uid').html($.fn.uuid());
    </script>
    ...
    <label id="uid"></label>

$.fn.stringToFunction

Returns a window function based on a string
