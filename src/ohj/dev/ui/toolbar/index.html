ohj.pageslider UI control b0.1

A container control to hold pages that can be navigated using slide animations.



Usage

Quick Start

    <script>
        $('#content').ohjpageslider();
    </script>
    ...
    <div id="content">
        <div id="page1">1</div>
        <div id="page2">2</div>
        <div id="page3">3</div>
        <div id="page4">4</div>
        <div id="page5">5</div>
    </div>

Options

Option Description Default
height The height of the page slider in css format 450px
speed The speed in milliseconds at which the page slides at 200
onload The callback to call when a page has been navigated to. null
onunload The callback to call when a page has been navigated away from. null

Public Methods

Method Description
navigateToPage(pageElement) Causes the page slider to navigate to a page using the slide animation
navigateBack() Causes the page slider to navigate back to the previous page
navigateNext() Causes the page slider to navigate to the next page defined within the DOM

Events

Event Description
onload(toPageElement) Evented when a page has been navigated to
onunload(fromPageElement) Evented when a page has been navigated away from
